Emacs LISP tools to:
Make reverse word-lists for all languages

Advantages
Elegant, easy to read (for some!)
Easy to integrate with built-in LISP interpreter in Emacs

Disadvantages
Hard to validate
"Mixed content" (in the SGML sense) difficult and tedious to represent
Not buzzword-compliant in current age of XML
